# Development execution on local environment.
1. Clone the repository.
2. Execute:
```
npm install
```
3. Install Nest CLI.
```
npm i -g @nestjs/cli
```
4. Run Docker DB
```
docker-compose up -d
```

## Used stack
* MongoDB
* Nest
